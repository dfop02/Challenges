1 # Desafio

Conclua a função no arquivo challenge.rb.

A função recebe uma string, e retorna a primeira letra não repetida.
Se todas as letras se repetem, retorna uma string vazia.
A função não faz distinção entre maiúsculas e minúsculas

examplp:
Input: "stress"
Output: 't'

Input: "sTreSS"
Output: "T".
